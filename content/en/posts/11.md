Latex in Telegram


I came up with the temporal solution for publishing texts with math expressions included. It doesn't allow to put formulas in the middle of the string and quite cumbersome to use, but it works. One can check my current approach.

The current approach:

𐂅Write an [article](<https://github.com/tarstars/telepub/blob/main/article.md>) in markdown
𐂅Use [agentic policies](<https://github.com/tarstars/telepub/blob/main/AGENTS.md>) to create html and pics
𐂅Use [template editor](<https://instantview.telegram.org/my/tarstars.github.io/?url=https%3A%2F%2Ftarstars.github.io%2Ftelepub%2Fgood_looking%2Findex.html>) to create IV link (probably this link requires my authorization)
𐂅Insert [link](<https://t.me/share/url?url=https%3A%2F%2Ft.me%2Fiv%3Furl%3Dhttps%253A%252F%252Ftarstars.github.io%252Ftelepub%252Fgood_looking%252Findex.html%26rhash%3D335bb2b74c586e>) into the message, it gives instant view button with preview
